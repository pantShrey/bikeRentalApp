# Bike Rental App

This is a sample project for bike rental app using Firebase.

## Features

- User authentication using Firebase
- Bike rentals data stored in Firestore
- Users can see a list of available bikes and rent one
- Owners can list their bikes



## Setup

- Clone this project
- Create a project in Firebase console and download config file
- Add the API keys from this file to firebaseConfig.ts
- Run `npm install`
- Run 'npm expo start' to start the expo client
